# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues, discussions, or pull requests.

Instead, report them privately using one of the following channels:

- [GitHub private vulnerability reporting](https://github.com/devletes/filament-timeline-view/security/advisories/new)
- Email: [packages@devletes.com](mailto:packages@devletes.com)

Please include as much of the following information as possible:

- The type of issue and its impact
- The affected version(s)
- Steps to reproduce, or a proof of concept

We aim to acknowledge reports within 72 hours and will keep you informed of our progress until the issue is resolved. We ask that you give us a reasonable disclosure window before publishing any details.
